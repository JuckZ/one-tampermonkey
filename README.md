# One Tampermonkey

## Usage

1. 安装[油猴插件](https://www.tampermonkey.net/)
2. 安装[One Tampermonkey](https://greasyfork.org/scripts/477102-onemonkey/code/OneMonkey.user.js)脚本并启用

## Features

- [x] 解除网页限制
- [x] 悬浮球(beta and just for me now)
  - [ ] 增加通用功能
- [x] 禁止用户数据采集(beta)
- [x] 密码查看
- [x] 元素间距标注
- [x] 展示隐藏内容
- [x] 网页自由编辑
- [d] 快捷搜索
- [d] 夜间模式
